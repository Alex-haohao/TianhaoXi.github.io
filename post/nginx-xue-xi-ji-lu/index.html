nginx学习记录

2020-08-16 1 min read

nginx 常用操作
nginx -v 查看版本
service nginx stop 关闭nignx
service nginx start 启动nginx
nginx -s reload 重启nginx

配置文件：
server {
#端口号
listen 80;
#域名或者ip
server_name 112.74.37.77;
#文件的路径
root /www/AlexMain;
#配置默认访问的页面
index index.html;
#配置代理
location /api {
proxy_pass http://127.0.0.1:3033;
}
location /apilist {
rewrite /apilist/(.*) /$1 break;
proxy_pass http://127.0.0.1:3033;
}

location / {
#当输入错误路径时，跳转到index.html页面
try_files $uri $uri/ /index.html;
}
}

nginx全局块： worker_processes auto; 并发数量

event块
events {
worker_connections 1024;
}
nignx 支持的最大连接数

http块：http全局块和server块

location ~ 表示uri 正则表达式 区分大小写
~* 正则表达式 不区分大小写
= 严格匹配

nginx 负载均衡
upstream myserver{}
轮询 ip_hash weight fair
